% Default figure sizing: HEIGHT-based, not width-based.
%
% All figures are produced at figsize=(10,10) — square. But figures
% with colorbars (2D plots, correlation matrices) have extended width
% (e.g., 12x10 effective) due to the colorbar axes. Setting size by
% width makes these figures smaller than 1D plots at the same nominal
% width fraction, because the extra colorbar width eats into the budget.
%
% Setting size by HEIGHT avoids this: a 0.45\linewidth height gives
% consistent visual sizing regardless of colorbars, because the plot
% area height is always 10 inches. Two figures side-by-side at
% height=0.45\linewidth will have matching plot areas even if one has
% a colorbar and the other doesn't.
%

\section{Phase 4c Observed Inference: Full
Data}\label{phase-4c-observed-inference-full-data}

\needspace{4\baselineskip}
\subsection{Summary}\label{summary}

Phase 4c applies the frozen Phase 4a/4b score-template model to all
available Run2012B/C TauPlusX data in
\texttt{phase3\_selection/outputs/sensitivity\_selected\_events.npz}. No
full-data retuning of the selection, classifier, categories, score bins,
or statistical model was performed. The Phase 4b human gate is recorded
as auto-passed by explicit user instruction.

The fitted observable remains
\texttt{mva\_score\_hist\_gradient\_boosting} in the \texttt{vbf},
\texttt{boosted}, and \texttt{zero\_jet} channels with bin edges
\texttt{{[}0.0,\ 0.2,\ 0.35,\ 0.5,\ 1.0{]}}. The Phase 4b score-template
warning is not removed: the 10\% combined data/MC ratio was
\texttt{1.232}, chi2/ndf was \texttt{1.732}, and the status was
\texttt{flagged} with flags
\texttt{\{\textquotesingle{}category\_chi2\_failure\textquotesingle{}:\ False,\ \textquotesingle{}combined\_chi2\_failure\textquotesingle{}:\ False,\ \textquotesingle{}combined\_ratio\_failure\textquotesingle{}:\ True\}}.

\needspace{4\baselineskip}
\subsection{W High-mT Control Scale}\label{w-high-mt-control-scale}

The full-data W+jets scale is derived only from the high-\texttt{mT}
control region using
\texttt{(N\_data\_CR\ -\ nonW\_MC\_CR)\ /\ W\_MC\_CR}. The full control
region contains \texttt{4023} data events, \texttt{W\_MC\ =\ 2414.781},
and \texttt{nonW\_MC\ =\ 1963.570}. The applied full-data scale is
\texttt{0.8528\ ±\ 0.0370} with status \texttt{valid}. The 10\% Phase 4b
scale was \texttt{0.8433\ ±\ 0.0868}.

\subsection{Phase 5 Obligation}\label{phase-5-obligation}

The Phase 5 paper must state that the score-modeling validation was
flagged in Phase 4b and remains a limitation of the simplified open-data
result. The result should not claim a clean CMS-quality score-template
model, nor should it describe the W+jets scale as signal-region tuned.
