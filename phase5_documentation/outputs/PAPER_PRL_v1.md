# A CMS Open Data Diagnostic Search for H to Tau Tau in the Mu Tau_h Final State

This PRL-formatted draft reports the audit-corrected reduced CMS Open Data
result. The optimized calibrated-score plus QCD/fake fit gives
`mu = 38.380 +7.035 -6.116`,
`mu < 50.000` at 95% CLs, with median expected
limit `mu < 1.974`. This optimized-score result fails the
observed validation gate (`fail`), so it is not retained as the
validated evidence result.

The previous visible-mass result is retained as the validated baseline:
`mu = 0.438 +4.944 -0.438`, `mu < 10.764` at 95% CLs, median expected `mu < 10.738`. The final comparison figures include both the optimized-score
attempt and this frozen baseline in the same signal-strength convention.
